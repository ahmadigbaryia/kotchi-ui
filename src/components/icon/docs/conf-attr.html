Here is a list of the available attributes for the <kui-icon> that can be used

Attribute Type Default value Description
kui-label String The icon's label - used also for screen readers
kui-icon Style The icon to be used from the available, For More information on the Font Awesome website
kui-size KUIIcon.Size The size of the icon, relative to the inherited font-size style value, For More information on the Font Awesome website
kui-fixed-width Boolean false Apply the fixed width style on the icon, For More information on the Font Awesome website
kui-list-icon Boolean false Replace default bullets in unordered lists, For More information on the Font Awesome website
kui-bordered Boolean false Adds a border arround the icon, For More information on the Font Awesome website
kui-pulled KUIIcon.PullDirection KUIIcon.PullDirection.Left Pull quotes or article icons, For More information on the Font Awesome website
kui-animate KUIIcon.Animate KUIIcon.Animate.Spin Makes the icon spin or have it rotate with 8 steps, For More information on the Font Awesome website
kui-rotate KUIIcon.Rotate Rotates the Icon with the given angel, For More information on the Font Awesome website
kui-flip KUIIcon.Flip Flips the Icon with the desired direction, For More information on the Font Awesome website
