Button <kui-button>

Use Kotchi UI custom buttons that are built upon Bootstrapâ€™s button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Attributes Configurations

Kotchi UI buttons are very flexable to configure and control

Here is a list of the available attributes for the <kui-button> that can be used

Attribute Type Default value Description
kui-text String The text of the button
kui-style Style Style.Default (kui-button-default) Applies a different look to the button
kui-outline Boolean false Applies the outline style to the button
kui-disabled Boolean false Applies the disabled state to the button
kui-size Size Size.Normal (no class needed) Applies a different size to the button

Basic Examples

With Kotchi UI you can easily create buttons with configurable predefined styles
Default Dangerous Success Warning Link 
                        
	<kui-button kui-style="kui-button-default">Default</kui-button>
	<kui-button kui-style="kui-button-primary" kui-text="Primary"></kui-button>
	<kui-button kui-style="kui-button-secondary" kui-text="Secondary"></kui-button>
	<kui-button kui-style="kui-button-info" kui-text="Information"></kui-button>
	<kui-button kui-style="kui-button-dangerous">Dangerous</kui-button>
	<kui-button kui-style="kui-button-success">Success</kui-button>
	<kui-button kui-style="kui-button-warning">Warning</kui-button>
	<kui-button kui-style="kui-button-link">Link</kui-button>

Outline buttons

In need of a button, but not the hefty background colors they bring? add the kui-outline attribute to remove all background images and colors on any button.
Default Primary Secondary Information Dangerous Success Warning 
                        
	<kui-button kui-style="kui-button-default" kui-outline="">Default</kui-button>
	<kui-button kui-style="kui-button-primary" kui-outline="">Primary</kui-button>
	<kui-button kui-style="kui-button-secondary" kui-outline="">Secondary</kui-button>
	<kui-button kui-style="kui-button-info" kui-outline="">Information</kui-button>
	<kui-button kui-style="kui-button-dangerous" kui-outline="">Dangerous</kui-button>
	<kui-button kui-style="kui-button-success" kui-outline="">Success</kui-button>
	<kui-button kui-style="kui-button-warning" kui-outline="">Warning</kui-button>

Disabled state

Make buttons look inactive by adding the kui-disabled boolean attribute to any <kui-button> element.
Default outline disabled Primary disabled 
                        
        <kui-button kui-style="kui-button-default" kui-outline="" kui-disabled="">Default outline disabled</kui-button>
	<kui-button kui-style="kui-button-primary" kui-disabled="">Primary disabled</kui-button>

Button Different Sizes

In need for different button sizes, no worries we have 4 different sizes for your convenient.
                        
	<style>
		kui-button[kui-size="kui-button-block"] {
			--margin-top: 10px;
			--margin-bottom: 10px;
		}
	</style>
	<kui-button kui-style="kui-button-default" kui-outline="" kui-size="kui-button-small" kui-text="Small Default outline button"></kui-button>
	<kui-button kui-style="kui-button-primary" kui-disabled="" kui-size="kui-button-large" kui-text="Large Primary disabled button"></kui-button>
	<kui-button kui-style="kui-button-info" kui-size="kui-button-block" kui-text="Full width button"></kui-button>
