Configurations and Attributes

Kotchi UI buttons are very flexable to configure and control

Here is a list of the available attributes for the uik-button that can be used

Attribute Type Default value Description
uik-text String The text of the button
uik-style Style Style.Default (uik-button-default) Applies a different look to the button
uik-outline Boolean false Applies the outline style to the button
uik-disabled Boolean false Applies the disabled state to the button
uik-size Size Size.Normal (no class needed) Applies a different size to the button
