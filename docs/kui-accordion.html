Attributes Configurations

Here is a list of the available attributes for the <kui-accordion> that can be used

Attribute Type Default value Description
kui-toggle Boolean Forces the accordion to toggle the visible section - keeping at most one section visible at a time
kui-collapse-icon String The icon to be displayed to collapse the section, it'll be rotated 180 degrees for expand icon
